APE.KritaTools v1.0.0

APE.KritaTools imports Zoo Tycoon 1 graphics into Krita for editing.

Features

  • Import ZT1 graphics
  • Override embedded palette
  • Supports background frames (isolate bg frame option available)
  • Option to import layers with solid background
  • Windows/Linux support

Usage

  1. Open Krita
  2. Go to the "Tools" menu
  3. Click on "Scripts" and then "Load APE Image into Krita"
  4. Choose the ZT1 image you want to import
  5. Accept default palette or uncheck "Use Embedded Palette" to select a new one.
  6. Import
